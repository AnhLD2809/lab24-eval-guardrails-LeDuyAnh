# Demo

Thư mục này dành cho artefact demo khi nộp bài.

Ưu tiên:

1. Đặt file thật tại `demo/demo-video.mp4`
2. Hoặc chèn YouTube unlisted link vào [README.md](../README.md)

Video nên thể hiện tối thiểu:

1. Chạy `Phase A` trên một vài câu hỏi đại diện
2. Minh họa `Phase B` pairwise judge
3. Minh họa `Phase C` với `PII`, `adversarial`, và `latency benchmark`
